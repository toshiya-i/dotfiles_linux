# dotfiles_linux
dotfiles for Linux

## Usage

```
% git clone git@github.com:toshiya-i/dotfiles_linux.git
% cd dotfiles_linux
% sh ./deploy.sh
% sh ./initialize.sh
➜  dotfiles_linux git:(master) ✗
Ctrl+d
% env zsh
```

initialize.shでoh-my-zshをインストールするとzshが起動して次の処理が進まないため、Ctrl+dでshellを抜けてください。
その後、再度env zshでzshを起動してください。
